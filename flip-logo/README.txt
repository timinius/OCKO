ФЛИП — ЛОГОТИП
================

Знак: буква Ф (φ) — кольцо из двух стрелок в обороте + вертикальная
стойка. Идея «флип»: обмен, перепродажа, переворот товара.

СОДЕРЖИМОЕ
----------
Логотип Флип.html      — бренд-лист: логотип, цвета, шрифт, применение
                         (откройте в браузере)

png/
  logo-light.png       — логотип + текст, на светлом
  logo-dark.png        — логотип + текст, на тёмном
  mark-green.png       — только знак, зелёный (прозрачный фон)
  mark-mint.png        — только знак, мятный (прозрачный фон)
  mark-white.png       — только знак, белый/моно (прозрачный фон)
  app-icon-1024.png    — иконка приложения (мятный знак на тёмном)
  favicon-256/64/32.png — фавиконы для вкладки браузера

svg/
  mark-green.svg       — векторный знак (для светлого фона)
  mark-mint.svg        — векторный знак (для тёмного фона)
  mark-white.svg       — векторный знак (моно)
